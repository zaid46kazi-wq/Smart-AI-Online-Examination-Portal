🐛 TROUBLESHOOTING

  Problem: "node is not recognized"
  ──────────────────────────────────
  Solution: Install Node.js from https://nodejs.org
            Restart terminal
            Run: node server.js

  Problem: "Module not found: compression"
  ─────────────────────────────────────────
  Solution: npm install compression express-rate-limit
            node server.js

  Problem: "Port 3000 already in use"
  ──────────────────────────────────
  Solution: taskkill /F /IM node.exe
            Or: set PORT=3001 && node server.js

  Problem: "Cannot connect to Supabase"
  ─────────────────────────────────────
  Solution: Check internet connection
            Check firewall
            Wait 10 seconds
            Refresh browser

  Problem: "Browser shows 'Connection refused'"
  ───────────────────────────────────────────────
  Solution: Wait 5-10 seconds after starting
            Try: http://localhost:3000 (not https)
            Check console: node server.js output
            Try different browser
            Clear browser cache: Ctrl+Shift+Delete
